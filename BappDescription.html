Galaxy (MITM Decryption) is a Burp Suite extension that aims to simplify and increase the efficiency of testing encrypted traffic, making it as straightforward as testing plain text traffic.\n\n

Features

\n
    \n
  • Automated traffic decryption: After setting up the custom hook, the plugin will automatically decrypt subsequent proxy traffic.
    • \n
  • Linked with security tools: Supports links with sqlmap and xray, helping you discover potential security vulnerabilities more efficiently.
    • \n
  • Supports multiple Burp modules: Applicable to numerous Burp modules, including Intruder, Proxy, Repeater, and Scanner.
    • \n
  • Additional features: Bypass Host Check, Bypass Auth of Path, Parse Swagger API Doc.
    • \n
\n\n

Usage

\n\nThe Galaxy extension is useful in the following scenarios:\n\n
    \n
  • During penetration testing, you find HTTP messages from the website are encrypted.
    • \n
  • The encryption logic is complex, involving a combination of encryption algorithms, custom algorithms, and dynamic keys.
    • \n
  • You want to use a scanner that scans plaintext requests and retrieves plaintext responses once the scan request is sent.
    • \n
  • You can reverse engineer the websites encryption and decryption logic (including invoking client code via hook). Basic coding skills are required.
    • \n
\n\nThe Galaxy extension can automatically decrypt the requests/responses after proxy, and send the decrypted requests to Repeater - this then results in plaintext responses. The extension can work with sqlmap and xray to scan plaintext requests and obtain the decrypted response.\n\n,
